# MyBank - Personal Finance Management App 🏦

**MyBank** to aplikacja webowa stworzona w ramach mojej pracy inżynierskiej. Jej głównym celem jest ułatwienie codziennego zarządzania finansami osobistymi w prosty i przejrzysty sposób. 

## 🛠 Technologie
* C# / ASP.NET Core
* Entity Framework
* Razor Pages
* SQL Server

## 🌟 Główne funkcjonalności
* **Bezpieczeństwo i autoryzacja:** Pełny system rejestracji, logowania i zarządzania kontami użytkowników.
* **Przejrzysty Dashboard:** Panel główny prezentujący kluczowe dane: aktualny stan konta, podsumowanie finansowe oraz 5 ostatnich transakcji.
* **Zarządzanie transakcjami:** Obsługa przelewów oraz możliwość definiowania wydatków i przychodów jako jednorazowe lub **cykliczne**.
* **Pełna personalizacja:** Użytkownik może dowolnie tworzyć i edytować kategorie wydatków/przychodów, dostosowując system do własnych potrzeb.
* **Logika biznesowa:** Zaimplementowana bezpieczna i niezawodna mechanika operacji na koncie.
